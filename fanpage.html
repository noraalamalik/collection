Taking it back to basics, student will engage in yesterday's Web design aesthetics to make a one page fansite using HTML. Each student should choose a topic and create the content for their first webpage. Whereas the subject matter is open for the student to select.The purpose of this project is to achieve an understanding of HTML structure, to use HTML syntax to create a webpage and to create external hyperlinks, insert images and text.
